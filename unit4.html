Description of Unit 4: Creating Digital Animation

You will then design, create and test your own animated product, in a similar way to how it is done in industry, and be introduced to the technology and techniques used by the professionals. You will be given a brief which will need to be fulfilled. You will also need to think about the creative aspect of the project as well as technical skills to use. You will review your completed animated product having obtained feedback from others.

A - understand the applications and features of digital animation products
B - design a digital animation product
C - create, test and review a digital animation product

LOA

Applications of digital animation - Applications and purpose of digital animation, including: different existing digitally animated products (e.g. TV programmes, films, computer games, music videos, advertisements, internet and mobile content, simulation, product development), the effect (e.g. evoke emotion, educate) they have on different audiences (e.g. age, gender, interest, need), etc; Features of digital animation - Features of computer-animated products, e.g.: type of animation, 2-D and 3-D models, image type (bitmap/photo and vector), frames per second.


LOB

What goes into the design? - Designs include: intended audience (age, gender, interests), purpose and requirements, storyboards containing panels which outline the main assets and which include some main panels that show how the assets combine, with list of any ready-made assets. Alternative ideas for the design should be included.


LOC

Creating the animation - Preparing assets: Gather ready-made digital assets; use graphic-editing software to create original assets; Import original and ready-made assets, etc; Animation-editing software - 2-D digital animation techniques.
